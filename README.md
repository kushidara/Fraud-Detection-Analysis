# Credit Card Fraud Detection & Risk Analytics

## Project Overview
This project analyzes credit card transactions to identify fraudulent activity using Machine Learning and Data Analytics.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-Learn
- Power BI

## Results
- Analyzed 284K+ transactions
- Built Random Forest fraud detection model
- Achieved 97% precision on fraud detection
- Created Power BI dashboard
